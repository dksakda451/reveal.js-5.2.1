为什么选择reveal.js?

  • 支持HTML和CSS的完全控制
  • 精美的过渡效果和动画
  • 支持Markdown和多媒体内容
  • 移动端友好的响应式设计
  • 简单但功能强大的演示工具
讲解reveal.js的主要优势和特点。强调其灵活性和易用性。

}

// 调用函数
const message = greeting('王康康');
